# Code prioritarie in Python

Esempi pratici di utilizzo delle **code prioritarie** in Python.

---

## Casi implementati

### 1. `code.py`
- Implementazione di una classe `Coda_prioritaria` che riproduce il comportamento della classe standard `queue.PriorityQueue`.  
- All’interno della classe vengono messi in evidenza alcuni aspetti interessanti, tra cui:  
  - differenze tra l’uso di `remove()` e `pop()` nella rimozione dell’elemento minimo da una lista;  
  - impiego di `enumerate` per scorrere gli elementi con indice.  
- Nel file è inoltre presente un esempio pratico di utilizzo della classe `queue.PriorityQueue`.
